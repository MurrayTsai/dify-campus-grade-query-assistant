# 如何导入到 Dify

## 1. 导入 DSL

1. 打开 Dify 控制台。
2. 进入 **Studio / 工作室**。
3. 点击创建应用，选择 **Import DSL / 导入 DSL**。
4. 上传本仓库中的文件：

```text
dify/workflow.yml
```

5. 导入完成后，进入工作流编排页面检查节点是否完整。

## 2. 重新配置插件

本项目使用了以下插件：

- DeepSeek 模型插件
- Database 插件
- ECharts 图表插件

导入 DSL 后，Dify 可能会要求你重新安装或授权这些插件。请根据页面提示完成配置。

## 3. 配置数据库连接

工作流中的 SQL Execute 节点使用 Database 插件执行 SQL。为了安全，本仓库没有保存真实数据库连接。

你需要在 Dify 中重新配置数据库连接，例如：

```text
mysql+pymysql://用户名:密码@数据库地址:端口/数据库名
```

请不要把真实连接信息提交到 GitHub。

## 4. 准备数据表

数据库中需要有 `student_grades` 表。字段说明见：

```text
docs/database_schema.md
```

## 5. 测试工作流

导入并配置完成后，可以用以下问题测试：

```text
查询全校各科目平均分情况
查询三年级一班语文成绩前五名
统计本学期各科及格率
展示各班数学平均分柱状图
查询不同学期英语平均分变化趋势
```

如果 SQL 生成正确但执行失败，优先检查：

1. 数据库连接是否正确。
2. 表名是否为 `student_grades`。
3. 字段名是否和 DSL 中的字段一致。
4. Database 插件是否已授权。
5. 当前数据库账号是否有 SELECT 权限。
