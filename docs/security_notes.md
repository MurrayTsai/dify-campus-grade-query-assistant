# 安全说明

本项目涉及自然语言转 SQL 和学生成绩数据，因此需要特别注意安全问题。

## 不要上传到 GitHub 的内容

请不要提交以下内容：

```text
.env
API Key
数据库地址
数据库用户名
数据库密码
真实学生数据
真实成绩表导出文件
真实学生姓名和学号
```

## SQL 安全建议

建议在 Prompt 和代码节点中增加限制：

1. 只允许生成 `SELECT` 查询。
2. 禁止生成 `INSERT`、`UPDATE`、`DELETE`、`DROP`、`ALTER`、`TRUNCATE` 等语句。
3. 禁止多语句执行，例如 `SELECT ...; DROP TABLE ...;`。
4. 数据库账号只授予只读权限。
5. 对结果行数添加限制，例如默认 `LIMIT 100`。

## 隐私建议

如果用于真实校园系统：

1. 普通用户不应查询他人成绩。
2. 学生姓名和学号应脱敏展示。
3. 查询日志不要记录完整隐私信息。
4. 面试作品集建议使用模拟数据，不使用真实校园数据。
